# BOS — Business Operating System

Unified frontend + backend for Almailam Engineering Consultants' practice management platform. Replaces the module-browsing model of the earlier `almailam-roadmap-ui` prototype with a due-date-driven Home: **Calendar** (what's due, what's overdue, what needs a decision) and **Workspace** (Projects, Clients, Documents, Government Center, Contracts, Quotations, Payments, Reports, Admin — full depth, unchanged in function).

Built as one tightly-connected system: a single shared Calendar/Escalation engine drives tasks, government submission deadlines, and contract milestones alike, and frontend + backend are documented and versioned together in this one repo.

Start at [`docs/00-Overview.md`](docs/00-Overview.md) for the full documentation set — vision, functional scope, data model, API reference, architecture, the calendar/escalation engine spec, security, performance, UI design system, coding standards, testing, deployment, the build-order pass plan, and the effort estimate.
